Tetris Q-Learning AI

Reinforcement learning agent for Tetris

Developed a Q-learning neural network that learns to play Tetris by evaluating board states and optimizing moves.

Java Reinforcement Learning Neural Networks Q-Learning

Project Details

Implemented a Q-learning agent that plays Tetris using a neural network to evaluate board states. The agent learns through reinforcement learning by optimizing for line clears while minimizing board height, holes, and unevenness. Features include custom reward function design, exploration strategies, and state feature engineering.

Key Features

  • Custom neural network architecture for Q-value approximation
  • Engineered state features including lines cleared, holes, bumps, and board height metrics
  • Dynamic exploration strategy that adapts during training
  • Custom reward function balancing immediate and long-term rewards
  • Experience replay and minibatch training

Java Reinforcement Learning Neural Networks Q-Learning Matrix Operations Feature Engineering
